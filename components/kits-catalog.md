# Kits Catalog  
**Genesis Network – All Flat-Pack Kits**  
**Last Updated: January 21, 2026**

All kits are manufactured in **tile factory pods** using **waste-derived toughened ceramics** (70–95% from C&D slag, rice husk ash silica, farm residues, ELFM/brownfield slag). They are designed for rapid self-replication, mobility, and 95–99% resource closure. Costs reflect mature-phase economies (automation, pre-supply); revenues include premium sales, upgrades, and credit sharing.
